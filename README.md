# AIScript
スクリプト言語をAIに作らせようとする試み。

## コンソール電卓 (Console Calculator)

Pythonで作成された対話型コンソール電卓。基本的な四則演算をサポートします。

### 使い方

```bash
python3 calculator.py
```

または、実行権限を付与して直接実行:

```bash
./calculator.py
```

### 機能

- 加算 (+)
- 減算 (-)
- 乗算 (*)
- 除算 (/)
- 括弧による優先順位制御
- 負の数のサポート

### 使用例

```
>>> 2 + 3
結果: 5

>>> (5 + 3) * 4
結果: 32

>>> 100 / 4
結果: 25.0
```

### 終了方法

`q`、`quit`、または `exit` を入力してください。
